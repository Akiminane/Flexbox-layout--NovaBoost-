# flexbox-layout

This project is a landing page built as part of a layout exercise focused on mastering **CSS Flexbox**.

The goal was to reproduce a page based on a strict set of design constraints, without using frameworks or CSS Grid.

---

## Project Goals

- Recreate a web page from a detailed design brief
- Respect fixed dimensions and layout constraints
- Use **Flexbox only** for layout and alignment
- Implement a fixed header and centered sections
- Handle common layout challenges (hero overlay, card alignment, galleries)

---

## Technologies Used

- HTML5
- CSS3
- Flexbox (exclusive)
- No frameworks
- No libraries
- No CSS Grid

---

##  Key Constraints

- Full-width header with inner content limited to **1600px**
- Main sections constrained to **1200px**
- Fixed heights for major sections
- Fixed-size service cards and gallery images
- Layout built entirely with Flexbox

---

## Page Structure

- Fixed header with navigation
- Hero section with background image and overlay
- Services section with three cards
- Gallery section with six images
- Contact section
- Footer

---

##   What This Project Demonstrates

- Solid understanding of Flexbox axes and alignment
- Proper separation between full-width containers and constrained content
- Ability to follow a design specification accurately
- Clean and readable HTML/CSS structure

---



##  Notes

This project was created for learning purposes and focuses on layout accuracy rather than responsiveness or advanced interactivity.
